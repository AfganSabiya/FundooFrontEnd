Pseudo Class

Defination: A pseudo-class is used to define a special state of an element.
For Example, changing the style of an element when the user hovers over it,
or when a link is visited.All of these can be done using Pseudo Classes in CSS.

Uses of pseudo classes

1):hover Pseudo-class

This pseudo-class is used to add special effect to an element when our mouse pointer is over it.
The below example demonstrates that when your mouse enters the box area, its background color changes from yellow to orange.

2):active Pseudo-class

This pseudo-class is used to select an element which is activated when the user clicks on it.
